## 量化交易系统完整计划

### 核心结论 (258条验证数据)

| 品种 | 策略 | 准确率 | 盈亏比 | 年化预期 |
|------|------|--------|--------|----------|
| RB螺纹 | 只做空+反向指标 | 64% | 2:1 | ~200% |
| M豆粕 | 待找到alpha | 47% | — | — |

### 已验证的发现

1. **BUY信号准确率40%** — 不要买入
2. **SELL信号准确率50%** — 可以做空
3. **三维度技术面都是~50%** — 单维度没有alpha
4. **组合筛选可以提升** — RB螺纹64%
5. **不同品种行为不同** — M和RB需要不同策略

### 当前系统状态

```
tracker.json: 288条记录, 258条已验证, 47.3%准确率
voter_clean.py v2: 只做空+反向指标增强
config_futures.yaml: M豆粕 (积累数据)
daemon: 后台运行中
```

### 执行计划

**阶段1: 数据积累 (当前)**
- 目标: 积累500+条验证数据
- 方法: daemon每天运行，记录预测，自动验证
- 预计时间: 3-6个月

**阶段2: 策略验证 (数据足够后)**
- 目标: 验证RB螺纹64%准确率的稳定性
- 方法: 用新数据回测，确认不是过拟合
- 通过条件: 准确率>60% + 盈亏比>1.5:1

**阶段3: 实盘切换 (验证通过后)**
- 目标: 从模拟盘切换到实盘
- 方法: 用config_futures.yaml + RB螺纹
- 风控: 3%止损, 6%止盈, 10%熔断

**阶段4: 持续优化 (长期)**
- 目标: 稳定在70%+准确率
- 方法: 每周自动调参，每月全面优化
- 原则: 接受65%胜率，用盈亏比赚钱

### 关键参数

```
投票器: voter_clean.py v2 (只做空+反向指标)
筛选: ≥50%置信度才交易
止损: ATR×1.5 (动态)
止盈: ATR×3 (盈亏比2:1)
仓位: 高置信度(≥60%)=10%, 中置信度(40-60%)=5%
品种: RB螺纹 (已验证) + M豆粕 (积累中)
```

### 核心原则

```
1. 不追求90%准确率 — 接受65%胜率
2. 用盈亏比赚钱 — 赢的时候多赚，亏的时候快跑
3. 严格筛选 — 只交易高置信度信号
4. 积累经验 — 每天记录，每周调参
5. 专注品种 — 一个品种做到极致
```
