### 策略核心逻辑

```
趋势确认条件 (满足≥5个信号):
1. MA多头排列 (MA5>MA10>MA20>MA60) → +2分
2. 均线向上倾斜 → +1分
3. RSI在40-60区间 → +0.5分
4. MACD柱向上 → +1分
5. 布林带中轨向上 → +1分
6. 成交量放大 >120% → +1分

做多: bull_score >= 5
做空: bear_score >= 5
```
