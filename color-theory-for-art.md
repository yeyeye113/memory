# 色彩搭配理论

## 色轮关系（数学驱动，非感觉）

| 方案 | 关系 | 效果 |
|------|------|------|
| **互补色** | 色轮对面 180° | 最强对比，焦点明确 |
| **类似色** | 相邻 30° | 和谐统一，氛围感 |
| **三等分** | 120° 等距 | 丰富平衡 |
| **分裂互补** | 基色 + 对面两侧邻色 | 有对比但不刺眼 |

## 色温系统

- **冷色（210°-270°）**：蓝、靛、青 → 后退感、空间深度、宁静
- **暖色（0°-60°）**：红、橙、金 → 前进感、温暖、焦点
- **冷暖交界**：用赭石/灰调过渡，避免直接撞色

## 饱和度阶梯（空间分层）

- **远景**：低饱和（灰调）→ 空间退远
- **中景**：中饱和 → 主体存在感
- **点缀/高光**：高饱和 → 视觉焦点

## 明度骨架（先定明暗再填色）

- 最暗 → 最亮要有 5 级以上清晰层次
- 不能灰成一片，对比要明确
- 月光场景：天空最暗 vs 光源最亮 = 焦点

## 60-30-10 法则

- **60%** — 主色（背景/大面积）
- **30%** — 辅助色（次要元素）
- **10%** — 强调色（点缀/焦点）

## 常见错误

1. 渐变过渡太快 → 中间色灰暗脏 → 加更多色阶，减缓过渡
2. 对比度不够 → 焦点不明确 → 加强最亮/最暗差距
3. 冷暖混在一起 → 画面脏 → 先分区再交界
4. 所有颜色同饱和 → 没有主次 → 用饱和度阶梯分层
